## Environments
- Unity Editor: Version 2021.3.6f1
- Visual Studio. Note: không nên dùng Visual Studio Code hoặc Jetbrains Riders
- Github Desktop
## Unity Project
- Tạo Folder *_GAME* chứa các files cho game đang làm
- Copy folder *_SDK* từ repository [RocketSgSdk](https://github.com/rocketsaigon/RocketSgSdk) và duyệt qua các tài nguyên trong đó.
